❓ 常见问题

安装和配置中遇到问题？先来这里找答案

ccswitch 是为 Claude Code 设计的一键切换工具，运行 cc switch deepseek 等命令即可将 Claude Code 的后端模型切换到国产大模型。
codex++ 是为 Codex / OpenAI 兼容工具链设计的国产 API 路由工具，自动拦截 HTTP 请求并重定向到国内 API 端点，无需修改代码。
国产模型对 Claude Code 的 tool-use 格式兼容程度不同。代码问答、代码审查等基础场景完全可用；文件编辑、终端命令执行等高级功能可能部分受限。建议国产模型用于日常编码辅助场景。
  • DeepSeek：platform.deepseek.com 注册获取
  • 通义千问：dashscope.aliyun.com 注册获取
  • 智谱 GLM：open.bigmodel.cn 注册获取
  • 百度千帆：cloud.baidu.com 注册获取
各平台通常提供免费额度，具体以官方政策为准。
这通常是因为 Node.js 或 Python 未正确配置环境变量。请确认已安装对应运行时，并将安装路径添加到系统 PATH 中。详细步骤请查看对应教程页的环境准备部分。
ccswitch 和 codex++ 均为本地工具，API Key 仅存储在本地环境变量中，不会上传到任何云端服务器。所有 AI 请求直接发往你所配置的 API 服务商，不经过第三方中转。
