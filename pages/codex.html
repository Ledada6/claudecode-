二、ccswitch 配置国产 API ⭐

ccswitch 是一个命令行工具，可以一键将 Claude Code / Codex 等工具的后端模型切换到国产大模型，无需手动配置环境变量。

安装 ccswitch

📦 前往 GitHub 下载 ccswitch

或通过 npm 安装：

# 通过 npm 全局安装
npm install -g ccswitch

# 验证安装
cc --version
