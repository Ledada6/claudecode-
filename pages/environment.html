一、Windows 系统环境搭建

安装 Node.js + 配置环境变量

下载安装：

  1. 访问 nodejs.org，下载 LTS（长期支持）版本
  2. 运行安装程序，一路点击 Next
  3. 关键步骤：在安装向导中，务必勾选 Add to PATH（添加到环境变量）

手动配置环境变量（如果安装时忘记勾选）：

  1. 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
  2. 在"系统变量"中找到 Path，双击编辑
  3. 点击"新建"，添加 Node.js 安装路径（默认 C:\Program Files\nodejs\）
  4. 点击确定保存

Windows 安装 Git 工具

Git 是代码依赖管理的必备工具：

  1. 访问 git-scm.com 下载 Windows 版
  2. 运行安装程序，建议保持默认选项
  3. 安装完成后，打开 PowerShell 验证：
    git --version

Windows 多版本 Python 安装与环境隔离

注意：Windows 可能预装 Python（Microsoft Store 版本），建议从官网安装独立版本。
  1. 访问 python.org 下载 Python 3.10+
  2. 安装时务必勾选 Add Python to PATH
  3. 验证安装：
    python --version

环境变量快速检测方法

打开 PowerShell（Win+R → 输入 powershell），分别执行以下命令：

# 检测 Node.js
node --version

# 检测 npm
npm --version

# 检测 Python
python --version

# 检测 Git
git --version

如果显示"不是内部或外部命令"，说明对应软件未正确添加到 PATH 环境变量，请重新检查安装步骤。

国内镜像源配置

npm 镜像源（解决 npm install 慢的问题）：

# 使用淘宝镜像（推荐）
npm config set registry https://registry.npmmirror.com

# 验证是否生效
npm config get registry

pip 镜像源（解决 pip install 慢的问题）：

# 临时使用清华镜像
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple 包名

# 永久配置
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

二、Mac 系统环境搭建

Intel / Apple 芯片 Mac 适配版 Node.js 安装

💡 Apple 芯片用户注意：请下载 macOS ARM64 版本的 Node.js， Intel 芯片用户下载 macOS x64 版本。
  1. 访问 nodejs.org 下载对应版本
  2. 运行 .pkg 安装包，按提示完成安装
  3. 验证安装：
    node --version
npm --version

Mac 终端基础权限设置

Mac 系统对终端权限限制较严，安装全局包时可能需要：

# 使用 sudo 安装全局包
sudo npm install -g 包名

# 配置 npm 全局路径（避免每次使用 sudo）
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'

Homebrew 基础使用（简化环境搭建）

Homebrew 是 Mac 最流行的包管理器，可以一行命令安装各种工具：

# 安装 Homebrew（如尚未安装）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 通过 Homebrew 安装 Node.js
brew install node

# 通过 Homebrew 安装 Git
brew install git

# 通过 Homebrew 安装 Python
brew install python

三、Linux 系统（Ubuntu / CentOS）

服务器一键安装命令

Ubuntu / Debian：

# 更新包列表
sudo apt update && sudo apt upgrade -y

# 安装 Node.js（方式一：apt 安装）
sudo apt install nodejs npm -y

# 安装 Node.js（方式二：使用 NodeSource — 获得更新版本）
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install nodejs -y

# 安装 Git
sudo apt install git -y

# 安装 Python
sudo apt install python3 python3-pip -y

# 验证
node --version && npm --version && git --version && python3 --version

CentOS / RHEL：

# 安装 EPEL 源
sudo yum install epel-release -y

# 安装 Node.js
sudo yum install nodejs npm -y

# 安装 Git
sudo yum install git -y

# 安装 Python
sudo yum install python3 python3-pip -y

# 验证
node --version && npm --version && git --version && python3 --version

服务器权限与全局环境配置

Linux 系统需要注意用户权限问题：

# 避免全局安装需要 sudo（推荐）
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'

# 将以下内容添加到 ~/.bashrc 或 ~/.zshrc
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

四、国产模型 API 接入

除了 OpenAI 和 Anthropic，国内也有多款优秀的 AI 模型可通过 API 调用。以下教程帮你用 cc switch 命令一键切换国产模型后端。

注意：以下 API 均需自行前往各平台注册账号、申请 API Key，本站不代注册、不代充值。
使用 cc switch 前，请确保已安装 cc 工具（npm install -g cc）。

注册 & 获取 API Key（三家通用说明）

模型平台地址说明
DeepSeekplatform.deepseek.com性价比高，价格低廉
通义千问阿里云百炼有免费额度
智谱 GLMopen.bigmodel.cnGLM-4-Flash 免费

用 cc switch 一键切换

获取 API Key 后，使用 cc switch 命令即可配置并切换到对应模型：

切换到 DeepSeek

cc switch deepseek
# 按提示输入你的 DeepSeek API Key

切换到通义千问

cc switch qwen
# 按提示输入你的阿里云 DashScope API Key

切换到智谱 GLM

cc switch glm
# 按提示输入你的智谱 API Key
💡 提示：cc switch 会自动帮你完成环境变量配置和模型切换， 一条命令搞定，无需手动设置 base_url。后续在 Claude Code 和 Codex 中直接使用即可。
